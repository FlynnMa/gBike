Welcome to gBike Pages.

This project is an android application based on my bleController project. You can find the project here Here is a video for your reference: http://v.youku.com/v_show/id_XMTMxODk2NTgwNA==.html

Software architecture

This software can be used to communicate with remote bleController, based on my private high security protocal. This android part software contains Android part and JNI part, JNI part implements the communication protocal. Android part implemented the bluetooth device managments and UI interface.

Creating projects quickly

If you prefer to the project you may use android-elipse to quick build and run it to have a brief short look.
